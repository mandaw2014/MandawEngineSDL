Building

1. Install auto-py-to-exe with
pip install auto-py-to-exe

2. In CMD or Terminal, type
auto-py-to-exe

3. Enter the script location

4. Select either One File or
One Directory

5. Select either Console Based
or Window Based

6. Select the icon, it has to
be a .ico file (optional)

7. Under Additional Files add
the following folders:
mandaw,
sdl2,
sdl2dll,
PySdl2-0.9.9.dist-info

Also add any other folders or
files that you used in your game

If you need help finding
site-packages, click here

8. Advanced (optional)

9. Click Settings and choose the
Output Directory

10. Finally Click Convert .PY to .EXE
.
